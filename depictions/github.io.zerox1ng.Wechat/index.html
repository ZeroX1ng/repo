介绍：Jan的微信插件，部分功能付费（文字转语音、MP3、MP4转语音等），特色功能语音包

兼容：iOS 12.0 - 14.7

官方源：公众号Jan分享

说明：1.5.9-6为老版本，1.0.3为全新重制版本（bug比较多）

Jan WeChat 1.0.7 更新内容 1. 修复分类语音包超过15个以上时出现分类重复； 2. 修复分类语音包超过15个以上时添加语音出现2个弹出； 3. 修复自动确认登录，适配 8.0.8； 4. 杂项功能新增上传长头像、自带勾选同步消息、显示并勾选登录该设备； 5. 消息功能新增自动接听语音、自动接听视频、自动拒接语音视频； 注意：自动接收视频需 App 在前台，不然摄像头无法工作。上传长头像需注意图片大小，太大容易失败。

1.0.4 更新记录 消息功能增加未接视频自动回复，未接语音自动回复； 新增菜单功能，微信主界面右上角 + 号可选增加清理未读消息、一键删除所有会话、解析短视频（无水印下载短视频）； 自动功能增加抢红包功能； 朋友圈功能增加朋友圈评论通知、朋友圈评论回复、朋友圈自动点赞、常驻后台； 新增 #选择 命令，恢复人物语音； 新增聊天界面语音增加到语音包； 恢复短视频链接转语音，部分短视频链接无法转语音，暂时无解； 新增文字 +1、语音 +1、表情 +1 菜单图标，文字转语音菜单图标，视频转语音、转发朋友圈菜单图标，语音试听、添加至语音包菜单图标； 授权版语音包图标自定义路径及文件名称：微信文档目录/Library/Caches/VoiceBao/Voice.png ，深色模式图标名称为 Voice_Dark.png，自定义语音包图标需放两个文件。

1.0.3 更新记录 兼容 libhooker 基板； 修复朋友圈转发闪退问题； 修复聊天界面长按文件、图片、语音闪退问题； 修复点歌功能，乱触发问题； 修复点击猜拳骰子会出现微信卡死或闪退问题； 修复发送语音包内语言，未回到聊天界面问题； 修复进群欢迎，无法选择加入的群问题； 删除 1-15 号人物，文字转语音之前请再次设置； 恢复语音试听功能； 新增自动功能，具体功能详见思维导图或下文叙述。
